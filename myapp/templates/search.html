Song Lyric Search

Search for your favorite song to see what genre the machine learning model thinks it is.

Due to restrictions with the site that hosts the lyrics this search is done by the text of the lyrics. This means it may not return the song you want, even if you enter the exact song name. If that happens, try to search using the first line or a line from the chorus.

Due to a lack of memory in the Heroku application a model built with 10,000 rows is used in this section.
